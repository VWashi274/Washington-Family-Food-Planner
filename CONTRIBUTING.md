# Contributing

Thank you for your interest in contributing to the Washington-Walker Family Food Planner!

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/your-feature-name`
5. Make your changes and test them with `npm run dev`
6. Build to verify: `npm run build`
7. Commit your changes: `git commit -am 'Add some feature'`
8. Push to your fork: `git push origin feature/your-feature-name`
9. Submit a pull request

## Development Guidelines

- Use TypeScript for type safety
- Follow the existing code style
- Test your changes with `npm run dev`
- Ensure the app builds without errors with `npm run build`
- Keep the warm color theme (#F7E8B4, #5C4033) consistent
- Use Playfair Display for headings, Poppins for body text

## Code Style

- Use 2 spaces for indentation
- Use semicolons at the end of statements
- Use camelCase for variables and functions
- Use PascalCase for React components
- Add comments for complex logic

## Reporting Issues

If you find a bug or have a feature request, please create an issue with:
- A clear description
- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Screenshots if applicable

## Questions?

Feel free to open an issue or contact the maintainers.
