# Washington-Walker Family Food Planner

A React + Vite app to help the Washington-Walker family select weekly meals.

## Setup

1. Run `npm install`
2. Run `npm run dev`
3. Open the local URL shown by Vite

## Features

- Assign meals to each day of the week
- Add new meals with categories, tags, and photos
- Save plans automatically in the browser
- Edit recipe notes directly in the app
- Upload meal images from your device
- Mark meals as favorites and remove meals
- Filter meals by favorites and category
- Pantry / refrigerator check to suggest meals you can make now
- AI-style alternative suggestions for different needs
- View a weekly plan preview
