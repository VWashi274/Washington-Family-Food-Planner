# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-13

### Added
- Initial release of Washington-Walker Family Food Planner
- Weekly meal planning with day-by-day assignment
- Protein filtering system with quick-select buttons
- Meal management (add, edit, delete, favorite)
- Image upload functionality for meals
- Recipe note editor
- CSV import with customizable settings
- Pantry check for meal suggestions
- AI-style alternative meal suggestions
- Category and protein-based filtering
- Settings toggle for CSV import visibility
- Local storage persistence for meals, schedule, and pantry
- Custom warm color theme with serif/sans-serif typography
- Inspirational verse display
- Responsive card-based UI design

### Features
- React 18.3.1 with TypeScript
- Vite 5.4.0 for fast development and production builds
- PapaParse for CSV handling
- React hooks for state management
- localStorage for persistent data

### Planned
- Ingredient lists and step-by-step recipe fields
- Grocery list and shopping planner
- Family member preferences and meal ratings
- Meal prep suggestions
- Nutritional information
- Recipe sharing and collaboration
- Mobile app version
